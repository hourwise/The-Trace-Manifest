# MVP Scope

## Objective

Prove that The Trace Manifest can collect, classify, explain, and answer questions from current AI information while preserving provenance and uncertainty.

## Included

### Public experience

- Homepage
- Intelligence feed
- Topic pages
- Story pages
- Daily briefing
- Basic model directory
- Basic benchmark registry
- Research feed
- GitHub radar
- Source pages
- Corrections ledger
- Methodology pages
- RSS
- Newsletter signup
- TRACE Guides only after six complete, reviewed guides are ready for a populated public section

### Curation

- Source registry
- Scheduled ingestion
- URL deduplication
- Basic semantic deduplication
- Story clustering
- Topic classification
- Source classification
- Evidence labels
- Manual review queue
- Correction workflow
- Administrator-only social discovery with separately evaluated linked-source candidates

### Ask box

- Limited public questions
- Citation-grounded answers
- Freshness date
- Confidence label
- Supporting and conflicting evidence
- Practical recommendation

### Administration

- Add, pause, and remove sources
- Review stories
- Edit summaries
- Approve high-impact claims
- Mark records corrected or superseded
- Pin important stories
- Inspect failed ingestion jobs

### Monetisation foundations

- Commercial-policy and disclosure schema only; no paid placement is live
- Privacy and consent framework

## Excluded

- Automated Reddit scraping
- Full social firehose
- User comments
- Public API
- Enterprise accounts
- Native applications
- Large historical archive
- Real-time minute-level updates
- Original large-scale benchmark execution
- Autonomous publication of high-impact claims
- Automatic publication until ADR 0010 gates and launch-stability criteria are met
- Paid editorial inclusion, paid rankings, behavioural advertising, or automatic affiliate-link insertion
- Complex team workspaces

## Governing decisions

The focused launch follows [ADR 0009](../adr/0009-governed-social-media-signal-intake-linked-source-discovery-and-outbound-linking.md), [ADR 0010](../adr/0010-expanded-editorial-scope-curated-products-and-governed-auto-publication.md), [ADR 0012](../adr/0012-durable-controls-access-admin-and-publication-boundaries.md), and [ADR 0013](../adr/0013-trace-guides-lab-authorship-verification-and-ask-trace-knowledge-integration.md). Commercial and public-sharing extensions remain governed by ADR 0011 and ADR 0014 respectively.
