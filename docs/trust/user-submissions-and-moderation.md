# User Submissions and Moderation

User-submitted links, questions, correction requests, and evidence are untrusted inputs.

## Prohibited content

- Illegal material
- Threats
- Harassment
- Doxxing
- Malware
- Spam
- Undisclosed promotion
- Impersonation
- Fabricated evidence
- Coordinated manipulation
- Prompt or retrieval poisoning

## Controls

- URL validation
- Sanitisation
- Rate limits
- Input limits
- Prompt-injection detection
- Retrieval isolation
- Citation requirements
- Human review for high-risk submissions
- Account restriction for repeated abuse

## Submission policy

Submitted content never becomes evidence automatically. All user-submitted material must be reviewed against primary sources before affecting evidence ratings, trust labels, or recommendations.
