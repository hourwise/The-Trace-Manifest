# SEO and Audience Strategy

Primary acquisition will come from organic search, newsletters, direct sharing, and repeat use.

## Early content pillars

- Best models for specific use cases
- Model and provider comparisons
- Local-model hardware guidance
- Benchmark interpretation
- AI agents and MCP infrastructure
- Open-weight licensing
- Pricing changes
- Current best-practice explainers

## Initial static decision pages

- Best local coding models by RAM
- Best models for structured output
- Best providers for UK and EU businesses
- Current MCP gateway comparison
- Best open-weight models for commercial use
- AI API price comparison

Every decision page must show its update date, methodology, evidence, limitations, affiliate disclosure, and change history.

## Content cadence

- Daily briefing
- 1–2 decision pages weekly
- Weekly market summary
- Monthly benchmark-health review
- Quarterly state-of-AI report
